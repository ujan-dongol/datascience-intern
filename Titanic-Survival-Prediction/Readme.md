# Titanic Survival Prediction

## Project Overview

This project focuses on building a machine learning model to predict whether a passenger survived the Titanic disaster.

The objective of this project is to demonstrate practical data science skills including data preprocessing, exploratory data analysis, feature engineering, and machine learning model development.

This project was completed as part of a Data Science Internship Program.

## Project Workflow

1 Data Collection  
2 Data Cleaning

4 Feature Engineering  
5 Model Training  
6 Model Evaluation  

---

## Model Used

Logistic Regression

The model was trained using Scikit-learn and evaluated using accuracy score and confusion matrix.

---

## Results

The model successfully predicts passenger survival using features such as passenger class, age, fare, and gender.
